#归并排序  
 
##自顶向下的方式  
要理解归并排序，要先理解merge方法,  
merge()方法:该方法先将所有元素复制到aux[]中，然后再归并回a[]中。方法在归并时，进行了4个条件判断:左半边用尽(取右半边的元素)，右半边用尽(取左半边的元素)，右半边的当前元素小于左半边的当前元素(取右半边的元素)以及右半边的当前元素大于等于左半边的当前元素(取左半边的元素)。   直白点，就是两个有序的数组，每次取最小的值放入原有数组当中，结束后，就是有序的了。  

现在再来看sort()方法，递归方法  
对数组a[lo...hi]先将它分成左右部分，不断递归，下潜，最后分解成对1个元素排序(1个元素是有序的)，到底部，然后向上，对2个元素排序然后有序，4个元素排序然后有序，8个....(通过调用merge方法)。  


##自底向上的方式  
思路:归并那些微型数组，然后再对归并子数组，直到我们将整个数组归并在一起。  
sz=1,sz=2,sz=4...不断递增。  

关键是理解:  
1.  外循环每次递增两倍长度,sz=sz+sz，每次内循环结束那么sz+sz长度是有序的(merge方法的性质)  
2.  内循环对规定的子数组进行排序，两两排序lo += sz+sz, 每次对sz+sz使用merge方法。
3.  还有注意边界值`merge(a,lo,lo+sz-1,Math.min(lo+sz+sz-1, N-1));` lo+sz-1是两个子数组的中间值，Math.min(lo+sz+sz-1, N-1)防止超出数组长度。  


归并排序的缺点就是:额外的空间和N成正比，空间不是最优的。  
优点:能够保证将任意长度为N的数组排序所需的时间和NlogN成正比。
