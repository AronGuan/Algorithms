#三向切分的快速排序   
目的:解决包含大量重复元素的情况  
思路:将数组切分为三部分，分别对应于小于，等于和大于切分元素的数组元素。  
方式:它从左到右遍历数组一次，维护一个指针lt使得a[lo..lt-1]中的元素都小于v,一个指针gt使得a[gt-1..hi]中的元素都大于v,一个指针i使得a[lt..i-1]中的元素都等于v，而[i..gt]的元素都还未确定。  

规则:  
1. a[i]小于v,将a[lt]和a[i]交换,将lt和i加一  
2. a[i]大于v,将a[gt]和a[i]交换，将a[gt]和a[i]交换，将gt减一  
3. a[i]等于v，将i加一。  

特点：每一次遍历数组，相比于快速排序，是把切分元素逐渐向右偏离，然后和与之相等的元素归为到一起，这样它们就不会被包含在递归调用的子数组中。  

最坏的情况：所有主键均不相同。  
当存在包含大量重复元素时，性能比归并排序好  
性能：介于N和NlogN之间
