#Rabin-Karp指纹字符串查找算法  
思路:基于散列的字符查找算法，需要计算模式字符串的散列函数，然后用相同的散列函数计算文本中所有可能的M个字符的子字符串散列值并寻找匹配。  等价于将模式保存在一张散列表中，然后在文本的所有子字符串中进行查找，这里关键是如何计算M个字符的子字符串散列值得方法，而Rabin-karp就是解决这个问题(需要预处理)  

计算散列函数  
使用Hornor方法，用于除留余数法计算散列值  
计算用char值数组表示的R进制的M位数的散列函数，所需时间与M成正比。
```
	//Horner方法，用于除留余数法计算散列值
	private long hash(String key,int M){
		//计算key[0..M-1]的散列值
		long h = 0;
		for(int j = 0;j < M;j++)
			h = (R*h + key.charAt(j))%Q;
		return h;
	}
```  

关键思想：对应所有位置i,高效计算文本中i+1位置的子字符串散列值。  
方式:减去第一个数字的值，乘以R,再加上最后一个数字的值，并且不需要保存这些数的值，只需要保存它们除以Q之后的余数。  

验证正确性  
蒙特卡洛:运行时间是线性级别且出错的概率极小(概率论保证)(通过使Q设为任意大的一个值)  
拉斯维加斯算法:能够保证正确性且性能极其接近线性级别。
